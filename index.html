Welcome to Simon Game

Simon is a fun and challenging memory game where players must recall and repeat a series of colored patterns. Each level introduces a new color to the sequence, testing your memory as you strive to beat your best score!

How to Play

  • Press "Start Game" to begin.
  • Watch the sequence of flashing colors.
  • Repeat the sequence by pressing the buttons in the same order.
  • If you succeed, the game will add another color to the sequence.
  • Continue until you make a mistake.
  • If you make a mistake, the game is over, and you can restart.
